Formalizing the Merge-on-Read resolution path, deletion indicators, and correctness theorems.

III. Formal Specification of the Read Path (Logical Row Projection)

Under Merge-on-Read, physical records must be dynamically reconciled with deleted rows at query execution time. This section establishes the algebraic correction formulas that query engines use to guarantee logical consistency.

Let $d \in D_k$ be a physical data file containing an ordered sequence of records, and let $I_d = [0, |d| - 1] \subset \mathbb{N}_0$ be the physical index space of the row offsets within $d$.

Record Locator Offset mapping
$$\text{Record}_d: I_d \to \text{Row}$$
A physical record selector function that points to a specific physical line number (index $i$) in data file $d$ to retrieve its fields.
Active Logical Rows
$$\text{ActiveRows}(d, S_k) = \{ \text{Record}_d(i) \mid i \in I_d \land \Psi(d, i, S_k) = 0 \}$$
The final set of logical rows returned to the client after filtering out all physically deleted records under state snapshot $S_k$.
Composite Delete Indicator Function
$$\Psi(d, i, S_k) = \mathbb{I} \left[ \Psi_{\text{pos}}(d, i, S_k) = 1 \lor \Psi_{\text{dv}}(d, i, S_k) = 1 \lor \Psi_{\text{eq}}(d, i, S_k) = 1 \right]$$
Evaluates to $1$ (logically deleted) if any of the three physical delete indicators (Positional, Deletion Vector, or Equality rule) matches the row offset $i$.

Where the individual physical delete indicators evaluate as:

  1. Positional Delete Indicator: $$\Psi_{\text{pos}}(d, i, S_k) = \mathbb{I} \left[ \exists f_p \in F_{p, k} \text{ s.t. } f_p.\text{path} = d.\text{path} \land f_p.\text{pos} = i \land S_{\text{data}}(d) < S_{\text{delete}}(f_p) \right]$$
  2. Deletion Vector (Roaring Bitmap) Indicator: $$\Psi_{\text{dv}}(d, i, S_k) = \mathbb{I} \left[ \exists v_d \in V_{d, k} \text{ s.t. } v_d.\text{target\_path} = d.\text{path} \land i \in \text{Bitmap}(v_d) \land S_{\text{data}}(d) < S_{\text{delete}}(v_d) \right]$$
  3. Equality Delete Rule Indicator: $$\Psi_{\text{eq}}(d, i, S_k) = \mathbb{I} \left[ \exists f_e \in F_{e, k} \text{ s.t. } \Pi_{A}( \text{Record}_d(i) ) = \Pi_{A}(r_{f_e}) \land S_{\text{data}}(d) < S_{\text{delete}}(f_e) \right]$$ where $A \subset \mathcal{U}_{\text{ID}}$ represents the set of equality field IDs, and $\Pi$ is the projection operator.
Theorem 1: Correctness of MoR Resolution
$$\text{ActiveRows}(d, S_k) \equiv \text{CoW}(d, S_k)$$
Theorem 1: The Merge-on-Read logical projection $\text{ActiveRows}(d, S_k)$ evaluates to the identical logical state as a Copy-on-Write rewrite $\text{CoW}(d, S_k)$ executed at snapshot $S_k$.
Proof of Theorem 1:
By definition, a Copy-on-Write operation at snapshot $S_k$ reads physical records and filters them eagerly using the deletion rules active in $S_k$. The resulting rewritten data file $d'$ contains: $$d' = \{ r \mid r = \text{Record}_d(i) \land i \in I_d \land \neg \text{MatchesDeletedRules}(r, S_k) \}$$ Let us analyze the deletion rules evaluation. The function $\text{MatchesDeletedRules}(r, S_k)$ returns true if and only if $r$ was deleted via any active deletion mechanism committed between $S_{i+1}$ and $S_k$:
  1. If a positional delete exists for $(d.\text{path}, i)$ committed at sequence $s_p \le s_k$, then since $d$ was created at $s_i < s_p$, the relation $S_{\text{data}}(d) < S_{\text{delete}}(f_p)$ holds. Thus, $\Psi_{\text{pos}}(d, i, S_k) = 1$.
  2. If a roaring bitmap contains offset $i$ for $d.\text{path}$ written at $s_v \le s_k$, then because $s_i < s_v$, the relation $S_{\text{data}}(d) < S_{\text{delete}}(v_d)$ holds. Thus, $\Psi_{\text{dv}}(d, i, S_k) = 1$.
  3. If a row matches an equality delete rule $f_e$ written at $s_e \le s_k$, then because $s_i < s_e$, the relation $S_{\text{data}}(d) < S_{\text{delete}}(f_e)$ holds. Thus, $\Psi_{\text{eq}}(d, i, S_k) = 1$.
Since the composite indicator $\Psi(d, i, S_k) = 1$ if and only if any of the above conditions evaluate to $1$, we have: $$\Psi(d, i, S_k) = 1 \iff \text{MatchesDeletedRules}(\text{Record}_d(i), S_k)$$ Therefore: $$\text{ActiveRows}(d, S_k) = \{ \text{Record}_d(i) \mid i \in I_d \land \Psi(d, i, S_k) = 0 \}$$ $$\equiv \{ \text{Record}_d(i) \mid i \in I_d \land \neg \text{MatchesDeletedRules}(\text{Record}_d(i), S_k) \} \equiv \text{CoW}(d, S_k) \quad \blacksquare$$
