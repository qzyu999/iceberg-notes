### Comment by @Tishj at 2025-07-13 19:23:28.000 UTC

> It would be great for folks working on this here and in other languages if some example files could be added to https://github.com/apache/parquet-testing 🙏🏻

There are parquet files with Variants in them here: https://github.com/duckdb/duckdb/pull/18224
As the description says, these are generated by apache/iceberg

Only notable difference is that they are rewritten to add the Variant logical type to the top level struct of the Variant column.
And because DuckDB doesn't support Timestamp Nanos TZ, those columns and values are also rewritten to Timestamp Micros TZ (which is not a lossless conversion)

I recommend to regenerate them: https://github.com/Tishj/iceberg/tree/generate_variant
Do note that they generated with uuid names, in my PR I've renamed them to be descriptive (for the most part)
